This migration preserves the visual language from the original site and keeps raw CSS files for reference.

## Preserved Theme Modes

- **Grass / light mode** (`style.css`):
  Uses white content boxes over `boschgrass.jpg`, dark borders, and blue links.
- **Library / dark mode** (`style2.css`):
  Uses black content boxes over `library.jpg`, white borders, and yellow links.
